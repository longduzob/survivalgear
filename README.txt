DESCRIPTION
-----------
SurvivalGear Pro est un thème Shopify complet et professionnel conçu spécifiquement 
pour les boutiques de matériel de survie, camping, et équipement outdoor.

CARACTÉRISTIQUES
----------------
✓ Design responsive mobile-first
✓ Mode sombre/clair automatique
✓ Mega menu avec images
✓ Filtres de produits avancés
✓ Kit Builder interactif
✓ Intégration apps tierces (avis, wishlist, devise)
✓ Optimisé pour la performance
✓ SEO-friendly
✓ Multilingue (FR/EN)

STRUCTURE DU THÈME
------------------
config/
  - settings_schema.json : Configuration du thème
  - settings_data.json : Données par défaut

layout/
  - theme.liquid : Layout principal

templates/
  - index.liquid : Page d'accueil
  - product.liquid : Page produit
  - collection.liquid : Page collection
  - cart.liquid : Panier
  - search.liquid : Recherche
  - page.contact.liquid : Page contact

sections/
  - header.liquid : En-tête
  - footer.liquid : Pied de page
  - hero-banner.liquid : Bannière hero
  - featured-collections.liquid : Collections en vedette
  - kit-builder.liquid : Constructeur de kit
  - featured-products.liquid : Produits en vedette
  - blog-posts.liquid : Articles de blog

snippets/
  - product-card.liquid : Carte produit
  - app-blocks.liquid : Blocs d'applications

assets/
  - theme.css : Styles principaux
  - theme-dark.css : Styles mode sombre
  - global.js : JavaScript global

locales/
  - fr.json : Traductions françaises

INSTALLATION
------------
1. Téléchargez le fichier ZIP du thème
2. Allez dans votre admin Shopify > Boutique en ligne > Thèmes
3. Cliquez sur "Ajouter un thème" > "Importer un fichier ZIP"
4. Sélectionnez le fichier ZIP téléchargé
5. Une fois importé, cliquez sur "Personnaliser" pour configurer

CONFIGURATION
-------------
Après installation, configurez :
- Logo et couleurs dans les paramètres du thème
- Menu de navigation
- Collections en vedette
- Produits en vedette
- Blog si utilisé
